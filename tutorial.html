pigeon: tutorial

pigeon is very easy to set up and begin using. It only takes one command, and then you can chat with it, how cool, am I right?!

Below is a basic quick start guide, what else you can do, and then some.

Getting started

First, invite pigeon to your server. You will need the Manage Guild permission as a baseline, and the same permissions pigeon requires. If not, ask someone higher.

Once you have invited pigeon, I recommend you create a new channel just for it. Since pigeon is designed to occupy and listen to an entire channel, unless you already have an unused throwaway channel, create one.

Once you have created / selected a channel to use, you can initiate it using /chat and append the target channel. Only people with Manage Server can do this. After that, pigeon is immediately available to use, and you're all set!

Managing your chat

pigeon keeps up to 200 messages per chat, and the oldest ones disappear once you reach 200. You can quickly reset and export/import your chat at any time. Read on...

Resetting your chat

There are two ways to reset your chat:

  • Run /clearhistory to immediately clear
  • Run /history clear:true to export your chat and clear when done

Exporting your chat

You can easily export your chat using /history. As stated above, if you append clear:true to the execution, it clears your chat after exporting it. pigeon will then generate a JSON file containing the messages and basic info about you and the server you used it in. It then uploads it, where it is kept for 3 days before automatic deletion. pigeon will then return a link to this website where you can view your transcript and also allow you to download the JSON it created.

Importing your chat

From the transcript viewer, go down to the bottom and find "View raw data". Right-click or long-press it and press "Save link as". Go back to Discord, then run /import and attach the JSON file. You cannot import other peoples transcripts, unless you edit the JSON to match you, but that would be a very petty thing to do.

Managing pigeon

Let's cut to the chase: pigeon can only be used in channels you designate it to, and anywhere in your server when manually called. It will not send random messages, randomly DM you, none of that. pigeon can easily be disconnected and paused. Read on...

Pausing pigeon from responding

Run /ignore with how long you want pigeon to pause. You can interrupt this by using /unignore.

Stop pigeon from responding

Run /remove. Only people with Manage Server can do this.

Change / check what channel pigeon listens to

Run /chat with nothing added to check what channel, and /chat with a channel selected to change.

That's pretty much everything! If you have further questions, feel free to ask! Or, browse the site further with the buttons below!
